# Helno-Meme-Coin

Helno Meme Coin smart contract using Solidity and Hardhat

## Installation

```shell
npx hardhat help
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat ignition deploy ./ignition/modules/Lock.ts
```
